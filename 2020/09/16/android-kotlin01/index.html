kotlin - 学习之data class 数据类

2020-09-16 移动端 102 字
android

说明

  • 创建一些只保存数据的类，在kotlin中标记为data数据类
  • 数据类必须满足以下要求
    • 主构造函数需要至少有一个参数
    • 主构造函数的所有参数需要标记为 val 或 var
    • 数据类不能是抽象、开放、密封或者内部的
    • （在1.1之前）数据类只能实现接口
data class User(val name: String)
